更新一下设备

发表于 2019-05-22

一次在大鲁家看他玩一个色彩鲜艳的游戏，我说这是啥啊，怎么这么像大菠萝……然后这果然就是大菠萝……然后就被嘲笑了两个礼拜……

突然意识到从小到大无论笔记本、台式机，用过的所有机器都是核显，虽然不热衷玩大型游戏，但是总觉得这样人生不够完整。这种念头一旦萌芽，晚上觉都睡不好，哈哈，找个学习的借口，咬牙决定拔草。

配置

  • CPU AMD 锐龙 5 2600 处理器
  • 主板 微星B450M MORTAR迫击炮
  • 显卡 蓝宝石（Sapphire）RX580 2048SP 8G D5 白金版 OC
  • 固态硬盘 西部数据（Western Digital）500GB SSD固态硬盘 M.2接口(NVMe协议) Black系列SN750
  • 机械硬盘 希捷(Seagate)2TB 256MB 7200RPM 台式机机械硬盘 SATA接口 希捷酷鱼BarraCuda系列
  • 内存 金士顿(Kingston) DDR4 2666 8GB 台式机内存 骇客神条 Fury雷电系列 两条
  • 电源 酷冷至尊（CoolerMaster）额定550W GX550游戏台式电源

踩坑

总的来说基本上也算是一次点亮，还算顺利，只是第一次装机，还是踩了不少坑。配件装的顺序略显混乱，主板就拆装了两三次；显卡装之前没掰掉机箱背板的小条差点弄坏……最后各种线往侧板里面一塞，眼不见心不烦。个人感觉安装顺序：CPU->内存->固态->主板进机箱固定（提前确认是否需要支撑铜柱）->风扇->机箱跳针、USB等->机械硬盘->电源及各种接线->显卡及显卡电源（注意背板一般都要掰掉一块露出各种接口）->整理线路盖上侧板->接通电源点亮。当然型号不同自然顺序也会有所调整。总的来说，不管安装还是拔线，千万不要大力出奇迹，不然钱包两行泪……
